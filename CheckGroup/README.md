## QQ群信息查询插件

一个QQ群信息查询插件，用于获取和显示指定QQ群的详细信息。

### 功能特点
- 通过命令快速查询QQ群信息
- 显示群名称、群号、群人数和人数上限
- 支持直接查询当前所在群或指定群号的群信息

### 安装指南
```bash
pip install aiohttp
```

### 使用说明
#### QQ机器人命令
- `[提醒词]开群 【群号码】` - 查询指定群号的群信息
- 直接使用`[提醒词]开群` - 查询当前所在群的群信息

### 注意事项
1. 插件依赖WebSocket连接与框架进行通信，请注意正确地在机器人的 config.json 中配置连接信息
